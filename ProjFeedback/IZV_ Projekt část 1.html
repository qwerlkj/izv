Test 4

Test datových typů pro kraje "JHM" a "PLK" přes funkci get_dict:

  • dostatek floatů (minimálně d a e) a integerů (sloupce pX)
  • nejsou tam typy typu object, protože ty jsou pro numpy naprosto nevhodné (viz přednáška)

Test 5

Test souřadnic (sloupec d a e) pro kraje "JHM" a MSK" přes funkci get_dict:

  • test, zda se jedná o floaty zda jejich zbytek po dělení 1 je kladné číslo (neztratili jste desetinnou čárku a správně ji převedli na tečku)

Test 6

Test formátování PEP8 - Python má jednoznačně definované, jak má vypadat kód. Často vám to napoví i nějakou špatnou konstrukci. V příštích částech bude kontrolováno striktněji, aktuálně je to pouze informativní.

Test 7

Test rychlosti - pouze pokud byl validní test 1 a test 4 (jinak byste dosáhli velmi rychlé implementace, protože jste třeba neřešili převod na floaty). Pokud jste se dostali pod čas 4 sekundy, dostali jste 2 body, jinak 1 bod (při splnění podmínek) Postup:

  • 5x
    • smazání cache
    • volání dd.get_dict(["JHM"])
  • pokud je první čas vykonávání funkce moc dlouhý, tak se započítá jen ten (protože jste zřejmě i přes výslovné uvedení v dokumentaci cachovali výstup), jinak se číslo zprůměruje.

Pro zajímavost přikládám ukázku, jak se dala vypočítat a zobrazit data pro graf.

fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(8, 6))

reasons = {
    1: "Přerušovaná žlutá",
    2: "Semafor mimo provoz",
    3: "Dopravními značky",
    4: "Přenosné dopravní značky",
    5: "Nevyznačena",
    0: "Žádná úprava"
}

def hide_items(x):
    r = x.copy()
    r[r == 0] = np.nan
    return r

regions = sorted(np.unique(data_source["region"]))
values = np.zeros((len(reasons), len(regions)))

for zid, (zcode, zname) in enumerate(reasons.items()):
    for kid, kname in enumerate(regions):
        values[zid, kid] = np.sum((data_source["region"] == kname) & (data_source["p24"] == zcode))

pos = ax1.imshow(hide_items(values), norm=LogNorm(vmin=1, vmax=1e5))
fig.colorbar(pos, ax=ax1, label="Počet nehod")

settings = dict(
    yticks=np.arange(len(reasons)),
    yticklabels=reasons.values(),
    xticks=np.arange(len(regions)),
    xticklabels=regions
    )

ax1.set(title = "Absolutně", **settings)

pos = ax2.imshow(100*hide_items(values / np.sum(values, axis=1).reshape(-1, 1)), cmap="plasma", vmin=0)
fig.colorbar(pos, ax=ax2, label="Podíl nehod pro danou příčinu [%]")
ax2.set(title = "Relativně vůči příčině", **settings)

plt.tight_layout()
